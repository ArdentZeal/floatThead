Data Attributes Overview

HTML5 data attributes can be used to configure and bind floatThead to tables without writing a single line of javascript.

FloatThead's data attribute API should be familiar to anyone who has used Bootstrap's data attribute API. Lazy initialization is also supported via data attributes. Keep in mind, that by default the plugin does not use lazy init - it loads on DOM ready.

Any configuration option that supports a non-function value can be used used as a data attribute config option. There are also a few options that can only be used as data-attributes.

Data Attributes API

API follows the following convention: The option name must be prefixed with data-floatThead-

For more detailed explanations of the options, see the options docs

Attribute Name Value Description
data-provide='floatThead' Enables data attribute API for table and binds floatThead to this element on DOM ready, unless used with lazy init attribute
data-floatThead-lazy='true' Lazily initializes this table on user interaction
data-floatThead-scrollContainer selector Closest parent of the table that matches selector will be used as the scrollContainer
data-floatThead-getSizingRow selector Selector used by IE to find the sizing row within the table
data-floatThead- I am not going to document the other options here. See main options docs for what is available.
