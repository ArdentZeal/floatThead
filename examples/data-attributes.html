Data Attributes Overview

HTML5 data attributes can be used to configure and bind floatThead to tables without writing a single line of javascript.

FloatThead's data attribute API should be familiar to anyone who has used Bootstrap's data attribute API. Lazy initialization is also supported via data attributes. Keep in mind, that by default the plugin does not use lazy init - it loads on DOM ready.

Any configuration option that supports a non-function value can be used used as a data attribute config option. There are also a few options that can only be used as data-attributes.

The data attribute API follows the following convention:
The option name must be prefixed with data-floatThead-
For example data-floatThead-zIndex='42'.
